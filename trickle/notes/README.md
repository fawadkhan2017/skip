# About Us Page

## Overview
This is a modern, responsive "About Us" page built with React and Tailwind CSS. It features a clean design with several key sections to effectively communicate the company's story and values.

## Components
- **Header**: Responsive navigation bar with mobile menu.
- **Hero**: Engaging top section with a headline, description, call-to-action buttons, and a contextual image.
- **Stats**: A grid displaying key company statistics with icons.
- **Values**: A section outlining the core company values using cards.
- **Footer**: Comprehensive footer with links and social media icons.

## Technology Stack
- React 18
- Tailwind CSS
- Lucide Icons (Static)

## Notes
- This project uses purely frontend technologies without any build steps.
- Styling is primarily handled through Tailwind utility classes.
- Custom theme variables are defined in the `index.html` file.
